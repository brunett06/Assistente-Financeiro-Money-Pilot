# Assistente Virtual Financeiro com IA

## Descrição
Este projeto foi desenvolvido como parte do desafio final da DIO.

A proposta é criar um assistente virtual financeiro que permita interação em linguagem natural, auxiliando o usuário no entendimento de conceitos financeiros e no registro de informações.

## Projeto principal - Pilot AI
Este repositório também está vinculado ao projeto **Pilot AI**, um assistente financeiro conversacional.

🔗 Link do projeto:
moneypilot.lovable.app/app

## Como funciona o Pilot AI
O usuário pode simplesmente escrever ou falar frases como:

- "Gastei 15 no almoço"
- "Recebi 2000 de salário"
- "Comprei um tênis por 300 em 3x"

E o sistema entende automaticamente:
- tipo da transação
- valor
- categoria
- parcelamento

## Tecnologias e conceitos
- Inteligência Artificial (IA generativa)
- Processamento de linguagem natural
- Simulação financeira
- Experiência do usuário (UX)

## Implementação neste repositório
- Simulação de assistente financeiro em Python
- Cálculo de juros compostos
- Interação via terminal
